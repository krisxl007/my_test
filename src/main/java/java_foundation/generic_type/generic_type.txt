泛型的本质： 参数化类型
泛型的作用： 代码复用, 减少运行期异常
泛型的擦除： 泛型是编译期的概念，编译器会生成具体类型给class文件，所以运行期没有泛型的概念

通用表达形式：
- T（Type）： 任意类型
- E（Element）： 集合中的元素类型
- K（Key）： 键
- V（Value）： 值

泛型的用法：
- 泛型接口
- 泛型类
- 泛型方法

泛型的边界：
- 上边界(Upper Bounds Wildcards): <? extends T>
    - 所传入的参数(?)必须是继承了T的类型

- 下边界(Lower Bounds Wildcards): <? super T>
    - 所传入的参数(?)必须是T的父类
